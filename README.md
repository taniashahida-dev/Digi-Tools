
# 🚀 DigiTools

## 📖 Description

DigiTools is a simple web app where users can explore different digital tools.  
Users can add tools to their cart, remove them, and see the total price.  
The design is clean, responsive, and easy to use on all devices.

---

## 🛠️ Technologies Used

- React.js  
- Tailwind CSS  
- JavaScript (ES6)  
- React Toastify  

---

## ✨ Features

### 🛒 1. Add to Cart  
- Users can add tools to the cart easily  
- If the same item is added again, it shows a warning  

### ❌ 2. Remove from Cart  
- Users can remove any item from the cart instantly  

### 💰 3. Total Price Calculation  
- The app automatically calculates the total price  

---

## 📱 Responsive Design

This project works on:

- Mobile 📱  
- Tablet 📲  
- Desktop 💻  

---

## 🔔 Extra Features

- Toast messages for actions (add, remove, payment)  
- Simple and clean UI  
- Fast performance
